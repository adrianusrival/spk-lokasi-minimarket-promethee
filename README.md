# spk-lokasi-minimarket-promethee
Sistem Pendukung Keputusan untuk menentukan lokasi minimarket menggunakan PROMETHEE berbasis web

## Fitur
- Login User
- Input data alternatif
- Proses perhitungan PROMETHEE
- Menampilkan Hasil ranking lokasi

2. Jalankan di localhost (XAMPP)
3. Akses melalui browser
